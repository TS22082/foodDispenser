Think it, build it.


Set up a dispensing and rebooting schedule by opening your terminal and typing:

crontab -e
30 6 * * * /home/user/programs/servoGo.py
0 19 * * * sudo reboot

Every time the system reboots itself you'll need it to restart the server for the web app. Open the terminal and type:

crontab -e
@reboot python3 /home/user/programs/wifiCatFeeder/app.py
